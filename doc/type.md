##  类型常量的定义

- const F = 'function'  函数 
- const O = 'object' 对象
- const A = 'array' 数组

- const N = 'number' 数字
- const R = 'regexp' 正则
- const NULL = 'null'  null
- const UNDEFINED = 'undefined'  undefined
